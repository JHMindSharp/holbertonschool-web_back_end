# NoSQL Project

## Description
This repository contains shell scripts and Python programs for the NoSQL project, focused on interacting with MongoDB.

## Requirements
- **Operating System**: Ubuntu 18.04 LTS
- **MongoDB**: Version 4.2
- **Python**: Version 3.7
- **PyMongo**: Version 3.10

## Project Structure

### Shell Scripts
- `0-list_databases`
- `1-use_or_create_database`
- `2-insert`
- `3-all`
- `4-match`
- `5-count`
- `6-update`
- `7-delete`

### Python Scripts
- `8-all.py`
- `9-insert_school.py`
- `10-update_topics.py`
- `11-schools_by_topic.py`
- `12-log_stats.py`
